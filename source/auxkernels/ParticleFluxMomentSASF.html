ParticleFluxMomentSASF

buildconstruction:Undocumented Class

The ParticleFluxMomentSASF has not been documented. The content listed below should be used as a starting point for documenting the class, which includes the typical automatic documentation associated with a MooseObject; however, what is contained is ultimately determined by what is necessary to make the documentation clear for users.

An auxkernel which computes higher degree and order angular moments of the uncollided flux in the Self-Adjoint Scalar Flux approach.
