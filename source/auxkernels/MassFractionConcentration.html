MassFractionConcentration

buildconstruction:Undocumented Class

The MassFractionConcentration has not been documented. The content listed below should be used as a starting point for documenting the class, which includes the typical automatic documentation associated with a MooseObject; however, what is contained is ultimately determined by what is necessary to make the documentation clear for users.

An auxkernel for the purposes of computing a radionuclide density from a mass fraction for post-processing. This is either a number density or a mass density.
