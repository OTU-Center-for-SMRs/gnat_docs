ParticleFluxMoment

buildconstruction:Undocumented Class

The ParticleFluxMoment has not been documented. The content listed below should be used as a starting point for documenting the class, which includes the typical automatic documentation associated with a MooseObject; however, what is contained is ultimately determined by what is necessary to make the documentation clear for users.

Computes the flux moments of the scattering source using the quadrature rule provided by the material system.
