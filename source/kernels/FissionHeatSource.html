FissionHeatSource

buildconstruction:Undocumented Class

The FissionHeatSource has not been documented. The content listed below should be used as a starting point for documenting the class, which includes the typical automatic documentation associated with a MooseObject; however, what is contained is ultimately determined by what is necessary to make the documentation clear for users.

A class which computes a fission heat production term for neutronics calculations coupled with heat conduction and/or thermo-hydraulics. The weak form is given by .
