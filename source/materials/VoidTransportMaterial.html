VoidTransportMaterial

buildconstruction:Undocumented Class

The VoidTransportMaterial has not been documented. The content listed below should be used as a starting point for documenting the class, which includes the typical automatic documentation associated with a MooseObject; however, what is contained is ultimately determined by what is necessary to make the documentation clear for users.

A void material for regions with no neutron interactions. Provides the neutron group velocity () for transport problems and forces maximum void stabilization for the SAAF scheme. The properties must be listed in decreasing order by energy.
