ADFVMassFractionNuclideDiffusion

buildconstruction:Undocumented Class

The ADFVMassFractionNuclideDiffusion has not been documented. The content listed below should be used as a starting point for documenting the class, which includes the typical automatic documentation associated with a MooseObject; however, what is contained is ultimately determined by what is necessary to make the documentation clear for users.

Computes the diffusion term for the isotope scalar transport equation: . This kernel expects a diffusion coefficient from the material system.
